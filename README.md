# Rising Compass -- Societal-Prose Provenance Log

Public, append-only, **hash-only** provenance for the societal-effects prose
generated by Rising Compass (https://risingcompass.net). The prose text never
appears in this repo. Each record commits only:

    sha256( "<table>:<id>\n<generated_at_iso>\n<model>\n<prose>" )

so a specific prose value can be proven later by revealing it and re-hashing --
without publishing the text now.

## Files

- `societal-prose.jsonl` -- cumulative, human-readable log (one JSON record/line).
- `batches/batch-<a>-<b>.jsonl` -- the immutable per-sweep batch that is OpenTimestamped.
- `batches/*.jsonl.ots` -- the OpenTimestamps proof anchoring that batch to Bitcoin.

## Record fields

`v` (schema version), `anchor_id`, `table`, `id`, `generated_at` (ISO 8601,
the sealed generation time), `model`, `sha256`.

## Verifying a prose value

1. Recompute the hash from the published `table`, `id`, `generated_at`, `model`
   and the revealed prose, using the recipe above.
2. Confirm it matches the `sha256` in the log.
3. `ots verify batches/<batch>.jsonl.ots` confirms that batch (and thus the
   hashes in it) existed no later than its Bitcoin block.

Bitcoin -- not this repo, not the database, not the author -- is the trust root.
